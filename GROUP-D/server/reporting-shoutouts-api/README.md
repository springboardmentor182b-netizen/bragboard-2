XD# Reporting Shoutouts API

This API allows:
- Employees to report shoutouts  
- Employees to view their reports  
- Admin to view all reports  
- Admin to resolve reports  

## Routes

### POST /report
Submit a new shoutout report  

### GET /my-reports
Employee view his reports  

### GET /admin/reports
Admin view all reports  

### PUT /admin/resolve/:id
Admin resolves a report
